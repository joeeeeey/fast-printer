# fastPrinter

A VS Code plugin to help insert print statements in different languages.

## Usage

选中要打印的内容, shift+command+;

![](https://upload-images.jianshu.io/upload_images/2674994-245ef3be8e6d62aa.gif?imageMogr2/auto-orient/strip)

## Support languages

- Javascript
- TypeScript
- Ruby
- Python
- Java
